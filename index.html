zADE Portal Help

About

The zADE Portal is the central access point for Ben's Big Iron Lab — a personal z/OS 3.2 homelab running ADCD z32a. Use the navigation bar or scroll to reach each section.

Sections

  • System Management — z/OSMF and Zowe web interfaces.
  • Documentation & Reference — IBM Docs, Redbooks, JCL library, and support links.
  • Tools — Apache Guacamole (remote desktop) and MinIO (object storage).
  • APIs — z/OS Connect EE sample endpoints with live status and copy buttons.
  • Repositories — Local GitLab instance and GitHub profile.

Service Status Indicators

  • Reachable — portal can reach the service.
  • Checking — status poll in progress.
  • Unknown — service did not respond conclusively.
  • Unreachable — service could not be contacted.

Hover any dot to see its current state. Use Refresh Status to re-poll all services.

Tips

  • Click any API endpoint URL to open it in a new tab, or use the copy button to grab the URL.
  • All service links open in a new tab and do not leave the portal.
  • The floating ↑ button appears after scrolling down — click it to return to the top.

Environment

  • System: ADCD z32a — z/OS 3.2
  • Portal: zade.mainframehome.net/zADE-Portal/
  • Served by: IBM Liberty on z/OS
